proxyServiceKey

var proxyServiceKey: String?

The com.github.manosbatsis.corbeans.spring.boot.corda.service.CordaNodeService key to use for obtaining an object mapper RPC proxy. A value of * results in a random choice between the available node services. If null, a non-RPC object mapper will be used
