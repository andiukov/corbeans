NodeParams

 class NodeParams

Configuration of a single node from a corbeans perspective. Includes information corresponding to an RPC user credentials and an CordaRPCClientConfiguration. The DEFAULTS are taken from CordaRPCClientConfiguration.DEFAULT and can be overridden using corbeans.nodes.default.xx in your application.properties
