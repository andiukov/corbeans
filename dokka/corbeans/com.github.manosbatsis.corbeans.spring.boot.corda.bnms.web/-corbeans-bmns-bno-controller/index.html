CorbeansBmnsBnoController

open class CorbeansBmnsBnoController : CorbeansBmnsBnoBaseController (source)

Exposes BNMS Operator (BNO) methods as endpoints for a single node or, if by overriding getRequestNodeName(), multiple nodes.

To use the controller simply extend it and add a @RestController annotation:

@RestController
class MyBmnsBnoController: CorbeansBmnsBnoController()
