CordaNodeController()

Exposes CorbeansBaseController methods as endpoints for a single node or, by overriding getRequestNodeName(), multiple nodes.

To use the controller simply extend it and add a @RestController annotation:

@RestController
class MyNodeController: CordaNodeController()
