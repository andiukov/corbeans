class CordaNodesController (source)

Rest controller with basic endpoints for multiple corda nodes. Supports multiple nodes by using the nodeName * path variable to construct a Service Bean name, then uses it as a key to lookup and obtain * a CordaNodeService for the specific node from the autowired services map.
